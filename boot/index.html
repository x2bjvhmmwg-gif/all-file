Bootstrap 5 완전 마스터

첫 화면 돌아가기

Container / 컨테이너

페이지 레이아웃의 기본 틀. 콘텐츠를 가운데 정렬하고 좌우 여백 제공.

<div class="container">내용</div>
<div class="container-fluid">전체 너비</div>
  • 픽셀 대응
    • sm ≥ 576px → 540px
    • md ≥ 768px → 720px
    • lg ≥ 992px → 960px
    • xl ≥ 1200px → 1140px
    • xxl ≥ 1400px → 1320px

Row / 행 & Column / 열

그리드를 사용하여 화면을 행과 열로 나눔.

<div class="row">
  <div class="col-12 col-md-6">왼쪽</div>
  <div class="col-12 col-md-6">오른쪽</div>
</div>
  • col-12 → 모바일 100%
  • col-md-6 → md 이상 50%

Display / 표시

  • d-block → 블록 요소
  • d-inline → 인라인 요소
  • d-inline-block → 인라인-블록
  • d-flex → Flex 박스로 변경
  • d-none → 숨김
<div class="d-flex justify-content-center">가운데 정렬</div>

Flex / 플렉스

  • 방향: flex-row, flex-column
  • 정렬: justify-content-start/center/end/between/around, align-items-start/center/end
  • 순서: order-0 ~ order-5
  • 감싸기: flex-wrap, flex-nowrap
<div class="d-flex flex-row justify-content-between align-items-center">
  <div>왼쪽</div>
  <div>오른쪽</div>
</div>

Spacing / 여백

  • m → margin, p → padding
  • t → top, b → bottom, s → start(left), e → end(right), x → 좌우, y → 상하
  • 값: 0~5 (0=0px, 1=4px, 2=8px, 3=16px, 4=24px, 5=48px)
<div class="mt-3 mb-2 px-4 py-2">내용</div>

Text / 텍스트

  • text-start → 왼쪽 정렬
  • text-center → 가운데 정렬
  • text-end → 오른쪽 정렬
  • text-uppercase → 대문자
  • text-lowercase → 소문자
  • text-capitalize → 첫 글자 대문자
  • fw-bold → 글씨 굵게
  • fw-semibold → 준볼드
  • fw-normal → 기본
<p class="text-center text-uppercase fw-bold">가운데 대문자 굵은 글자</p>

Background / 배경색

  • bg-primary, bg-secondary, bg-success, bg-danger, bg-warning, bg-info, bg-light, bg-dark, bg-white
  • text-white, text-dark
  • bg-opacity-25/50/75/100 → 배경 투명도
<div class="bg-success bg-opacity-75 text-white p-3">내용</div>

Images / 이미지

  • img-fluid → width:100%, height:auto
  • rounded → 테두리 둥글게
  • rounded-circle → 원형
  • object-fit-cover → 박스에 맞게 이미지 자르기
<img src="example.jpg" class="img-fluid rounded object-fit-cover" />

Buttons / 버튼

  • btn → 기본 버튼
  • btn-primary, btn-secondary, btn-success, btn-danger, btn-warning, btn-info, btn-light, btn-dark
  • btn-lg, btn-sm → 크기
  • disabled → 비활성화
<button class="btn btn-success btn-lg">클릭</button>

Forms / 폼

  • form-control → input, select, textarea 기본 스타일
  • form-check → 체크박스/라디오
  • form-label → label
  • form-text → 설명 글
  • is-valid / is-invalid → 검증 상태 표시
<input type="text" class="form-control" placeholder="입력">

Navbar / 네비게이션 바

  • navbar → 기본 navbar
  • navbar-expand-lg → lg 이상에서 확장
  • navbar-dark / navbar-light → 색상
  • bg-primary / bg-dark 등 → 배경색
  • navbar-brand → 로고 또는 사이트 이름
  • navbar-nav → 메뉴 목록
  • nav-item → 메뉴 항목
  • nav-link → 메뉴 링크
<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
  <a class="navbar-brand" href="#">Logo</a>
</nav>

Card / 카드

  • card → 카드 컨테이너
  • card-body → 내용 영역
  • card-title → 제목
  • card-text → 설명
  • card-img-top → 카드 상단 이미지
<div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top">
  <div class="card-body">
    <h5 class="card-title">제목</h5>
    <p class="card-text">설명</p>
  </div>
</div>

Alert / 알림

  • alert → 기본 알림
  • alert-primary, alert-success, alert-danger 등 → 색상
  • alert-dismissible → 닫기 버튼
<div class="alert alert-success alert-dismissible fade show">
  성공 알림
  <button type="button" class="btn-close" data-bs-dismiss="alert"></button>
</div>

Badge / 배지

  • badge → 배지 기본
  • bg-primary, bg-success 등 → 색상
  • rounded-pill → 동그란 모양
<span class="badge bg-success rounded-pill">New</span>

Table / 테이블

  • table → 기본 테이블
  • table-striped → 줄무늬
  • table-bordered → 테두리
  • table-hover → 마우스 호버
  • table-sm → 작은 테이블
<table class="table table-striped table-hover">
  ...
</table>

Modal / 모달

  • modal → 모달 기본
  • modal-dialog → 모달 박스
  • modal-content → 모달 내용
  • modal-header → 상단 제목
  • modal-body → 본문
  • modal-footer → 하단 버튼
<div class="modal fade" id="exampleModal">
  <div class="modal-dialog">
    <div class="modal-content">
      ...
    </div>
  </div>
</div>

Dropdown / 드롭다운

  • dropdown → 드롭다운 컨테이너
  • dropdown-menu → 메뉴
  • dropdown-item → 메뉴 아이템
  • dropdown-toggle → 버튼/링크
<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" data-bs-toggle="dropdown">메뉴</button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">항목1</a></li>
  </ul>
</div>

Pagination / 페이지네이션

  • pagination → 페이지 네이션
  • page-item → 페이지 항목
  • page-link → 링크
  • disabled, active → 상태 표시
<ul class="pagination">
  <li class="page-item active"><a class="page-link" href="#">1</a></li>
</ul>

Progress / 진행바

  • progress → 진행바 컨테이너
  • progress-bar → 진행 상태
  • bg-success, bg-danger 등 → 색상
  • style="width: 25%" → 진행 비율
<div class="progress">
  <div class="progress-bar bg-success" style="width:25%">25%</div>
</div>

Spinner / 로딩

  • spinner-border → 원형
  • spinner-grow → 점점 커짐
  • text-primary / text-success → 색상
<div class="spinner-border text-primary" role="status"></div>

Collapse / 접기

  • collapse → 접기 영역
  • show → 펼쳐진 상태
  • data-bs-toggle="collapse" → 버튼으로 접기
<a data-bs-toggle="collapse" href="#demo">열기</a>
<div id="demo" class="collapse">내용</div>
