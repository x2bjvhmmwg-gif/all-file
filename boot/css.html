CSS 완전 마스터

첫 화면 돌아가기

Colors / 색상

텍스트나 배경의 색상을 설정

  • color → 텍스트 색상
  • background-color → 배경 색상
  • transparent → 투명
p { color: white; background-color: #333; }

Text / 텍스트

글자 크기, 굵기, 정렬 등

  • text-align: left/center/right → 정렬
  • font-weight: normal/bold → 글자 굵기
  • text-transform: uppercase/lowercase/capitalize → 대소문자 변환
p { text-align: center; font-weight: bold; text-transform: uppercase; }

Box Model / 박스 모델

요소의 크기, 여백, 테두리 관리

  • margin → 외부 여백
  • padding → 내부 여백
  • border → 테두리
  • width / height → 너비 / 높이
div { margin: 10px; padding: 20px; border: 1px solid #fff; width: 200px; height: 100px; }

Flex / 플렉스

배치와 정렬을 손쉽게 제어

  • display: flex → Flex 박스로 설정
  • flex-direction: row/column → 방향
  • justify-content: flex-start/center/end/space-between → 가로 정렬
  • align-items: flex-start/center/end/stretch → 세로 정렬
  • flex-wrap: wrap/nowrap → 감싸기 여부
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

Grid / 그리드

2차원 배치 시스템

  • display: grid → Grid 설정
  • grid-template-columns → 열 정의
  • grid-template-rows → 행 정의
  • gap → 행/열 간격
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 10px;
}

Position / 위치

요소의 위치 지정

  • position: static/relative/absolute/fixed/sticky
  • top, right, bottom, left → 위치 조정
  • z-index → 쌓임 순서
div {
  position: absolute;
  top: 50px;
  left: 100px;
  z-index: 10;
}

Transition / 트랜지션

속성 변화에 애니메이션 효과 적용

  • transition-property → 변화 속성
  • transition-duration → 지속 시간
  • transition-timing-function → 속도 곡선
div {
  transition: all 0.3s ease-in-out;
}
div:hover {
  background-color: #555;
}
