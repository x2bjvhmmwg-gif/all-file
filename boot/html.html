HTML 완전 마스터

첫 화면 돌아가기

Headings / 제목

페이지에서 제목을 정의하는 태그들

  • <h1> ~ <h6> → 제목 크기 (h1 가장 큼, h6 가장 작음)
<h1>가장 큰 제목</h1>
<h3>중간 제목</h3>

Paragraph / 문단

텍스트 단락을 나타내는 태그

  • <p> → 일반 문단
<p>이것은 문단입니다.</p>

Links / 링크

다른 페이지나 위치로 이동하는 하이퍼링크

  • <a href="URL"> → 링크 연결
  • target="_blank" → 새 탭에서 열기
<a href="https://example.com" target="_blank">예시 링크</a>

Images / 이미지

웹 페이지에 이미지를 표시

  • <img src="이미지경로" alt="설명"> → 이미지 삽입
  • width, height → 이미지 크기 지정
<img src="example.jpg" alt="예시 이미지" width="200">

Lists / 목록

순서 있는 목록과 없는 목록

  • <ul> → 순서 없는 목록
  • <ol> → 순서 있는 목록
  • <li> → 목록 항목
<ul>
  <li>항목1</li>
  <li>항목2</li>
</ul>

<ol>
  <li>첫번째</li>
  <li>두번째</li>
</ol>

Tables / 테이블

데이터를 표 형태로 표시

  • <table> → 테이블 생성
  • <tr> → 행(Row)
  • <th> → 헤더 셀
  • <td> → 일반 셀
<table border="1">
  <tr><th>이름</th><th>나이</th></tr>
  <tr><td>홍길동</td><td>25</td></tr>
</table>

Forms / 폼

사용자 입력을 받는 태그들

  • <form> → 폼 컨테이너
  • <input type="text"> → 텍스트 입력
  • <input type="submit"> → 제출 버튼
  • <textarea> → 여러 줄 입력
  • <select> → 선택 박스
<form>
  <label>이름:</label>
  <input type="text">
  <input type="submit" value="제출">
</form>

Semantic Elements / 시맨틱 요소

의미를 가지는 HTML5 구조 요소

  • <header> → 헤더 영역
  • <footer> → 푸터 영역
  • <main> → 주요 콘텐츠
  • <section> → 구획
  • <article> → 독립적인 콘텐츠
  • <nav> → 내비게이션
<header>헤더</header>
<main>본문</main>
<footer>푸터</footer>
